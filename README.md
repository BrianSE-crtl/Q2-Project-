# Q2-Project-

A fun game that I recreated for a High School Project 

## Description

The player starts the game up and is met with a menu screen with 2 options (Play The Game or Credits), once they play, they are then playing an idle game. 

## Getting Started

### Dependencies

* python
* time module 
* os module
* threading 
* global 

### Executing Program

* In your IDE, make sure you are in the project folder in the terminal and type:
```sh
python Q2-PROJECT-.py
```
